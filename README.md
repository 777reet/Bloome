A modern wellness & mental health platform with interactive mindfulness tools. Features responsive design and therapeutic games for promoting focus, relaxation, and personal growth.

## Features
- **Interactive Memory Game**: Therapeutic card-matching game with wellness tips and calming effects
- **Responsive Design**: Mobile-first approach optimized for all device sizes
- **Theme Switching**: Dark/light mode toggle for user comfort and accessibility
- **Heart Interactions**: Spread positivity with interactive heart animations
- **Dynamic Forms**: Smart contact forms with real-time validation and feedback
- **Floating Animations**: Gentle background elements creating a calming atmosphere
- **Accessibility First**: WCAG-compliant with screen reader support and keyboard navigation
- **Wellness Content**: Curated mindfulness tips and motivational messages

## Tech Stack
**Frontend:**
- Pure HTML5 with semantic markup and accessibility features
- CSS3 with Grid, Flexbox, and custom properties for modern styling
- Vanilla JavaScript (ES6+) with smooth animations and progressive enhancement
- Google Fonts integration (Poppins, Playfair Display, Dancing Script)

**Planned Backend:**
- Python FastAPI with automatic API documentation
- PostgreSQL database with Redis caching layer
- JWT authentication for secure user sessions
- RESTful API with JSON responses for wellness data

**Features:**
- Mood tracking with analytics and progress insights
- Personalized wellness recommendations
- Community forums and peer support systems
- Professional therapist integration portal

## Usage
1. Clone the repository: `git clone <repository-url>`
2. Open `index.html` in a modern web browser
3. Grant permissions for optimal interactive experience
4. No build process required - pure HTML/CSS/JS implementation
